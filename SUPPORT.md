# Support

- **Questions, bugs, and feature requests:** open a
  [GitHub issue](https://github.com/tpt-solutions) on the relevant
  repository using the provided issue forms.
- **Community chat:** join the [Discord](https://discord.gg/rC7Ss7UVu) for
  general discussion. This is not monitored for bug reports — use an issue
  for those.
- **Security vulnerabilities:** see [SECURITY.md](SECURITY.md) — do not
  report these as public issues.

There is no email or paid support tier at this time.
